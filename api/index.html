Parameter Description Type Required
tenant Name of tenant String TRUE

Request Body

Example

{
    "domain": "spidermonkey.com",
    "verified": true
}

Successful Reponse

Status: 201

Sample (Click to view) 
HTTP/1.1 201 Created
{
    "domain": "spidermonkey.com"
}
